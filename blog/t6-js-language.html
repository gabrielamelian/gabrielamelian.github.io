Some more basic Javascript

This week we have a look at some of the cornerstones of writing code: the use of parentheses, braces, brackets, and single and double quotes. They are used to tell the program to interpret things a certain way, much like punctuation does in written language. Below are all of the case uses that I can think of these on Javascript.

Parentheses ( )

These are used in a way that is similar to maths in the sense that you can use them to tell the computer to do some action first before proceeding to the next. The difference is that it not only applies to mathematical operations, but also logical and others. It is telling the program "do this first, then proceed with the rest".
(6 + 6) + "is the result of the operation"

Following this they are used in loops to express a condition that needs to happen for the loop to continue or in a simple "if" to proceed with the content of the expression.
if (winner === true){ lib.displayMessage('You win!'); }

Parentheses also define the arguments that will be passed to a function. This is done when we "call" the function as well. They way to write this is to put the name of the function first and then the parenthesis with whatever the information we want it to use. When we have a function that does not use any arguments we still add them but this time empty.
generateBoard();
function countSurroundingMines (cell) { }

Brackets [ ]

These are used to define arrays and to contain the things in them.
var cells = ["cell1", "cell2"]

They are also used to index parts of the array in order to find them from outside of it. All elements on an array have an order number starting from 0 so if we want to get the second object on our array we call it like this:
cells[1]

In a similar way to arrays, brackets are also used to find a property for an object, only in this case the property name needs to be in quotes or double quotes (string).
board = {cells: "black"}
board["cells"]

Braces { }

Braces are used to define Objects in Javascript as in the previous example
board = {cells: "black"}

They also wrap functions and loops when expressing the action to be done.

Single quotes ' ' and double quotes " "

They are used to define the limits of a string. You can use one of the other but not a combination of both.
