Is there such a thing as too much reflection?

What is grid based design?

My understanding is that this a convention used to create easily customizable structures for the elements of a website using a certain number of columns. This kind of design should also help the user to understand and navigate easily through the different elements.

Whats all the hype about responsive webpages?

The cool thing about this new trend is that you would be able to create a single webpage that can look good on lots of different devices ditching the good old mobile pages and other high maintenance stuff. I mean, how cool is that right?

What is semantic structure?

On HTML this means using the elements provided in the structure they were intended to so they have internal coherence. I believe the equivalent would be to use the correct structure to write a newspaper article for example: you wouldn't put the title in the middle of a paragraph and so forth.
